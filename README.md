# 高中教材资源库 📚

![Update Status](https://img.shields.io/badge/更新-进行中-brightgreen)

## 🌟 项目介绍

本仓库致力于收集、整理并免费分享**中国大陆义务教育阶段主流教材PDF版本**
部分文件因大小过大上传到Releases内，例如生物
目前已整理完成的教材包括但不限于：

✅ **高中信息技术** 必修+选择性必修  
✅ **高中通用技术** 必修+选择性必修（凤凰出版社）  
✅ **高中化学** 必修+选择性必修  
✅ **高中数学** 必修+选择性必修（A版/B版双版本）  
✅ **高中物理** 必修+选择性必修  
✅ **高中生物** 必修+选择性必修（浙江科学技术出版社）  
✅ **高中英语** 选择性必修  
✅ **高中语文** 选择性必修（上册/中册/下册）  

**持续更新中**：义务教育阶段教材、高中其他科目、最新版教材...

## 🤝 如何贡献

本仓库欢迎各位教育工作者、志愿者贡献教材资源！

### 贡献方式

1. **提交新教材**  
   - 将PDF文件按本仓库目录结构整理好  
   - 或直接放置在根目录，运行 `classifier.py` 自动归类  
   - 提交 Pull Request

2. **补充缺失册次**  
   - 查看仓库缺少哪些册次  
   - 上传对应PDF文件

3. **报错反馈**  
   - 发现文件损坏、命名错误、分类错误  
   - 在 Issues 中提交反馈
